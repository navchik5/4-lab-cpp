## 1 задача
Вычислить 	площадь 	треугольника. 
Результат должен быть типа double. 

### Алгоритм решения:

1. Создать класс RightTriangle
  - Два приватных поля: double a, double b (катеты)
2. Реализовать три конструктора:
  - По умолчанию: a = 1.0, b = 1.0
  - С параметрами: принимает a и b, вызывает сеттеры для проверки
  - Копирования: копирует значения из другого объекта
3. Сделать геттеры:
  - getA() - возвращает значение катета a
  - getB() - возвращает значение катета b
4. Сделать сеттеры:
  - setA(a) - проверяет что a > 0, иначе выводит ошибку и ставит 1.0 (значение по умолчанию)
  - setB(b) - проверяет что b > 0, иначе выводит ошибку и ставит 1.0 (значение по умолчанию)
5. Реализовать метод calculateArea():
  - Возвращает (a * b) / 2.0 - площадь прямоугольного треугольника
6. Перегрузить оператор вывода <<:
  - Выводит строку: "Треугольник с катетами: a = X, b = Y"

## 2 задача
Унарные операции: 
++ увеличивает стороны треугольника в 2 раза (чтобы вновь получить катеты прямоугольного треугольника); 
-- уменьшает стороны треугольника в 2 раза (чтобы вновь получить катеты прямоугольного треугольника); Операции приведения типа: double (явная) – результатом является площадь треугольника, если треугольник существует и отрицательное число в противном случае; bool (неявная) – результатом является true, если треугольник с такими длинами сторон существует и falseв противном случае. 
Бинарные операции: 
<= RightTriangle t– сравнивает площади треугольников; >= RightTriangle t - сравнивает площади треугольников. 

### Алгоритм решения:

1. Создание базового класса
  - Объявить класс RightTriangle с двумя приватными полями: a и b (катеты)
  - Реализовать три конструктора:
  - Конструктор по умолчанию (устанавливает a=1.0, b=1.0)
  - Конструктор с параметрами (использует сеттеры для проверки)
  - Конструктор копирования (создает копию объекта)
2. Реализация методов доступа к данным
  - Создать геттеры getA() и getB() для чтения значений катетов
  - Создать сеттеры setA() и setB() с проверкой корректности:
  - Если значение ≤ 0, вывести ошибку и установить 1.0
  - Иначе установить переданное значение
3. Реализация основного функционала
  - Реализовать метод calculateArea() по формуле: (a × b) / 2
4. Перегрузка унарных операторов
  - operator++() - увеличить оба катета в 2 раза
  - operator--() - уменьшить оба катета в 2 раза
5. Реализация операторов приведения типа
  - operator double() - если треугольник существует, вернуть площадь, иначе -1
  - operator bool() - вернуть true если оба катета > 0
6. Перегрузка бинарных операторов
  - operator<=(other) - сравнить площади треугольников
  - operator>=(other) - сравнить площади треугольников
7. Реализация оператора вывода
  - operator<< - вывести информацию о треугольнике в формате "Треугольник с катетами: a = X, b = Y"
